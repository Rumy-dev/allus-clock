# Allus Focus v0.2.9 - macOS

Pasta de preparação da versão atual para macOS.

## Build esperado

- Apple Silicon: `out/make/dmg/arm64/Allus Focus.dmg`
- Intel: `out/make/dmg/x64/Allus Focus.dmg`

## Como gerar no Mac

```bash
npm ci
npm run make
```

## Arquivos base desta versão

- `package.json`
- `forge.config.ts`
- `assets/info.plist`
- `assets/entitlements.plist`
- `assets/icon.icns`
