## Como instalar

Baixe a versão mais recente aqui: **[github.com/Rumy-dev/allus-clock/releases/latest](https://github.com/Rumy-dev/allus-clock/releases/latest)**

Na página de downloads, procure o arquivo do seu sistema:

| Sistema | Arquivo |
|---|---|
| Windows | `Allus.Focus-x.x.x.Setup.exe` |
| macOS | `AllusFocus.dmg` |

### Windows

1. Baixe o arquivo `...Setup.exe`.
2. Dê dois cliques nele. Se o Windows mostrar um aviso ("O Windows protegeu o computador"), clique em **Mais informações** → **Executar assim mesmo**.
3. O app abre e já fica pronto pra usar — não precisa configurar nada.

### macOS

1. Baixe o arquivo `AllusFocus.dmg`.
2. Dê dois cliques nele pra abrir, e arraste o ícone **Allus Focus** pra pasta **Applications**.
3. Abra o **Applications** e dê dois cliques em **Allus Focus**.
4. Como o app ainda não é assinado pela Apple, na primeira vez o macOS vai bloquear com "não é possível verificar o desenvolvedor". Pra liberar:
   - Clique com o botão direito (ou Control + clique) no ícone do **Allus Focus** → **Abrir** → confirme **Abrir** de novo na janela que aparecer.
   - Isso só é necessário uma vez.

## Primeiro acesso

O Allus Focus usa conta de time — não existe cadastro dentro do app. Peça pro administrador criar seu acesso e te passar o e-mail e senha. Depois de entrar, se esquecer a senha, use **"Esqueci minha senha"** na tela de login.

## Atualizações

O app se atualiza sozinho: sempre que uma versão nova é publicada, ele baixa e avisa quando reiniciar for necessário. Não precisa baixar o instalador de novo — isso só é preciso na primeira instalação.
